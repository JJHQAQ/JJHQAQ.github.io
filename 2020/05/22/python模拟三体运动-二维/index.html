使用技术

使用python的numpy模块，以及配套的matplotlib,其实只是个简单的数学画图工具，但是用不断刷新的办法可以类似达到动画的效果。

几个用到的函数

  • ion()

是matplotlib里的函数，意思是开启交互模式。

  • clf()

清空屏幕

  • plot()

画图函数，里面的参数很多，可以用它来画自定义的曲线或者点，可以设置大小，颜色等。

  • axis()

设置观测轴范围

  • show()

打印图形

具体实现

其实没啥技术含量，就是通过不断的计算每一个小时间点的加速度，从而得到球体下一个时刻的位置，然后不断打印。
