主席树练习题1(远古补题)

发表于 2020-07-06 | 分类于 ACM/ICPC | | 本文总阅读

前言

害，懒癌发作了就是没办法，一个月前做的题，半个月才补，然后今天回来做的总结。。。。离谱

题目大意

给一棵树，树有边权，多次询问，每组询问给定X(起点)，Y(终点)，C,K.
求最小的P,使得路径上小于C-P的边和大于C+P的边的数量小于等于K.
原题链接

解决

首先可以做一个主席树，可以做到logn得查询路径上比某个值小的边权的个数，然后就对P做个二分答案，就可以过了。
