前言

上周小学期的课是要求写一个比较完整的网站，前端后台都要有。刚好很久之前就有研究一点点html和css，刚好借此机会尝试一下能写成什么样。
用到的东西主要有：

  • html
  • css
  • javascript
  • php
  • mysql

HTML

HTML是一种标记语言，也就是说，html并不能用来做高深的算法问题，比起计算机语言，他跟文学意义的那个语言的功能更相近一点。html的存在，就是为了告诉浏览器，你的网页的内容以及显示结构是怎样的。

在HTML中，网页的内容(文本，图片等等)都是被包含在标签中的，标签的格式大概是<标签名>网页内容<\标签名>
HTML的标签大多数都需要有几个结束标签，也有一些不需要内容的单联标签不需要结束标签也可以正常使用。
标签与标签之间可以嵌套。
HTML5之后，世界的网页html标准固定化，以后的所有网页都将支持以往的版本，也就是说，将来的HTML5只会添加新标签或者为旧便签添加新的使用方法，而不会修改旧标签的旧方法。

CSS

CSS应该是在html之后出现的东西，他的出现，是为了使得网页的样式和标签分离。CSS出现之后，html就只负责网页的结构和内容，网页具体长什么样，就全都交给CSS控制了。
所谓样式，就是网页元素(标签)的具体表现形式，比如位置，颜色，动画效果，大小等等。

MYSQL

MYSQL是一个数据库软件，内部可以永久得保存数据信息，支持添加，查找，删除，修改等等。
mysql的软件基本架构，大概是这个样子
用户登录
↓
数据库（多个）
↓
表单多个
↓
数据

不同用户的登录有不同的操作权限，mysql中可以有多个数据库，每个数据库中有很多表单，表单中的数据就类似一个表格，表格有一个键值组，用来表示每条数据有哪些信息。

比如说键值是（姓名，性别，年龄）
那么表中的数据就是（JJH,男,三岁）这个样子。
